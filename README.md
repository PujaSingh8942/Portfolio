# Portfolio

A modern, responsive portfolio website showcasing my skills, projects, and professional background. Built with React and Tailwind CSS.

## Features

- **Hero Section** — Introduction and overview of who I am as a developer
- **About Section** — Information about my background, technologies I work with, and my work approach
- **Skills Section** — Complete list of technical skills and expertise
- **Projects Section** — Showcase of my featured work and projects
- **Contact Section** — Multiple ways to get in touch with me
- **Navigation Bar** — Smooth, responsive navigation throughout the site
- **Footer** — Links and additional information

## Tech Stack

- HTML
- CSS
- JavaScript
- React
- Tailwind CSS


## Getting Started

Clone the repository and install dependencies:
```bash
npm install
```

Run the development server:
```bash
npm run dev
```
